# 🚀 Modern Portfolio – Next.js 15 + Bun + tRPC

A high-performance, modern portfolio built with the latest technologies:

- **Next.js** (App Router, Turbopack) v15.0.3
- **tRPC**
- **Tanstack Query**
- **Bun**
- **TailwindCSS**
- **TypeScript**
- **Shadcn**
---

# 📦 Getting Started

## 🛠️ Prerequisites

- [Bun](https://bun.sh/) (v1.2 or later)

## 📥 Install Dependencies

```bun
bun install
```

## Development

```bun
bun dev
```

## Deployment

```bun
bun build
```
